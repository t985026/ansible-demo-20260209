1. Templates (Jinja2)

功能與用途

  • 使用變數動態生成配置檔案
  • 支援條件判斷 (if/else) 與迴圈 (loops)
  • 自動替換主機相關資訊 (Facts)

常用標籤

  • 變數: {{ var }}
  • 控制: {% if ... %}
  • 過濾器: var | default('val')

範例 (nginx.conf.j2)

server {

  listen {{ http_port }};

  server_name {{ server_name }};

  {% if enable_ssl %}

  listen 443 ssl;

  {% endif %}

}
